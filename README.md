# Test Slug - Next-Blog Integration

This repository was created to test and explore the [`@supergrowthai/next-blog`](https://www.npmjs.com/package/@supergrowthai/next-blog) package.

##  What is Next-Blog?

**Next-Blog** is a modern blogging solution built specifically for Next.js (App Router only). It aims to provide a seamless way to integrate a fully functional blog dashboard into your Next.js application — similar to how NextAuth simplifies authentication.

### 🔗 For more information, check out the [npm package page](https://www.npmjs.com/package/@supergrowthai/next-blog).
